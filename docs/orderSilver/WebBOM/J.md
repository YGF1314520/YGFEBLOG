# Web BOM 浏览器对象模型

## 什么是浏览器对象模型(Browser Object Model)

- **browser:浏览器** 
- **object:对象**  
- **model:模型**  

**BOM(Browser Object Model)** 是指**浏览器对象模型**，是用于**描述这种对象**与**对象之间层次关系**的**模型**，**浏览器对象模型**提供了**独立于内容**的、可以与**浏览器窗口进行互动**的对象结构。

**BOM由多个对象**组成，其中**代表浏览器窗口**的**Window对象**是**BOM的顶层对象**，**其他对象**都是**该对象的子对象**。
